# automacao-rh-n8n
Automação de RH desenvolvida com n8n, OpenAI, Google Sheets e Gmail. O fluxo recebe solicitações de colaboradores, utiliza IA para classificar o tipo de solicitação e definir a prioridade, registra os dados em uma planilha e envia notificações automáticas por e-mail. O objetivo é otimizar o atendimento inicial do RH e reduzir tarefas manuais.
